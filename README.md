# laravel-docker-sample

Laravel動かす用のDocker環境です

## How to use

1. `mv laravel-docker-sample YOUR_PROJECT_NAME`
2. `cd YOUR_PROJECT_NAME`
3. `cp env-example .env`
4. .envに値を書き込む
5. laravelのプロジェクトをディレクトリルートに配置します
6. `docker-compose up -d`
